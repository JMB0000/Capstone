# OBA Supplies - React UI (minimal scaffold)

This is a minimal React scaffold implementing the login and admin dashboard mockups.

Quick start (Windows PowerShell):

```powershell
cd "c:\Users\johnm\OneDrive\Desktop\Capstone"
npm install
npm start
```

Notes:
- The login form accepts any email/password for demo and navigates to the dashboard.
- To customize branding, replace the logo text in `src/components/Login.js` and `src/components/Dashboard.js`.
- This project uses `react-scripts` so it behaves like a standard Create React App project.

Files added:
- `package.json` - dependencies and scripts
- `public/index.html` - app host
- `src/index.js`, `src/App.js` - app entry and router
- `src/components/Login.js`, `src/components/Dashboard.js` - main UI
- `src/index.css` - styling
